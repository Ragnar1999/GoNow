Two sibling services share one contract: the frontend (React + Vite on :5173) calls the backend's `/api/*` REST endpoints, which in turn proxy all EGD GraphQL requests server-side to keep the token out of the browser. The same backend also exposes `POST /api/chat`, where an agent loop forwards tool-calling decisions to OpenRouter and executes predefined EGD tools (`search_player`, `get_player_details`, `compare_players`) against `egd_client.py`. The root Makefile is the single entry point that creates the Python venv, installs npm deps, and launches both servers via `make dev`/`make stop`; environment variables for EGD tokens and OpenRouter keys live exclusively in `backend/.env`.
